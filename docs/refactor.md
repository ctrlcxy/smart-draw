# 项目重构文档
需要对项目的UI和交互进行一次深度的重构，新的UI使用shadcn组件库。

## 要点
1. 页面采用drawio渲染界面占满整个布局，chat界面悬浮在drawio渲染界面的上层，由于右边，可以收起成一个悬浮按钮
2. 移除code editor组件，chat界面采用连续对话的设计，即上面是对话气泡区域，下面是一个输入框
3. chat输入框内最下面是一排tool，包含一个“+”可用于上传文件和图片，包含一个“发送”
4. 使用function call的形式来调用大模型，通过function call完成drawio的xml代码填入，因此需要参考现在的提示词和相关大模型调用逻辑进行改造

## 完成后的检查
1. 检查已经被替换了的功能，移除无用的代码
2. 检查数据调用链是否完整
